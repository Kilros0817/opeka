Avatar Button

Компонент Кнопка Аватара для отображения пользовательских аватаров с адаптивными изображениями. Поведение как радиокнопка: только одна кнопка в паре может быть активной одновременно.

Размеры: sm (74×101px), md (106×144px). Цвета: tr-white, tr-gray.

Props: size — размер (sm, md), bg_color — цвет фона (tr-white, tr-gray), src_mobile — URL изображения для мобильных устройств, src_desktop — URL изображения для десктопа, src_main — основной URL изображения (PNG fallback), alt — альтернативный текст для изображения, active — класс для активного состояния (avatar-btn--active).
