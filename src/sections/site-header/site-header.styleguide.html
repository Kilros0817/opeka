Site Header

Шапка с белым фоном, скругления 12px. Один HTML, адаптив через BEM-модификаторы --desktop / --mobile + медиа-запросы (mobile ≤767px).

Адаптивные модификаторы

Эти классы живут в исходной разметке всегда. Что они делают — не зависит от вёрстки страницы; CSS сам скрывает/показывает на нужной ширине:

Cheat sheet 
--desktop  показывается на ≥768px, скрыт на mobile:
      .site-header__top-list--desktop  служебные ссылки в top-баре
      .site-header__nav--desktop       основная навигация (Меню + chips)
      .site-header__tool--desktop      кнопка «Личный кабинет»
      .site-header__cta--desktop       CTA с полным текстом «Оставить заявку»

    --mobile   показывается на ≤767px, скрыт на desktop:
      .site-header__cta--mobile        CTA с сокращённым текстом «Записаться»
      .site-header__tool--mobile       бургер-кнопка (триггер mobile-меню)

Mobile-превью: уменьши окно браузера до <768px или открой DevTools Device toolbar (Cmd+Shift+M). Витрина рендерит шапку в её дизайнерской ширине 1496px с горизонтальным скроллом во frame.
